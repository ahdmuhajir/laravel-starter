# Create Laravel Project With Usefull Package

## Laravel Admin-LTE3

```sh
$ laravel new project-name
$ composer require jeroennoten/laravel-adminlte
$ composer require laravel/ui
$ php artisan ui:controllers
$ php artisan adminlte:install
```
